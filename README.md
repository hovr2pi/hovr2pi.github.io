install.hovr2pi.org
=================

This website contains  a collection of scripts to build various
software packages.

Usage
------------

```bash
$ \curl -l http://install.hovr2pi.org/modulename | bash
```

Available Modules
------------
* columbus
* git
* homeshtick
* linxbrew
* wgetarx
